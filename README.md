MERN Task Manager Application ||
 A full-stack MERN Task Manager application built with React, Node.js, Express, and MongoDB. This application helps teams and individuals manage tasks efficiently with role-based access, dashboards, reports, and calendar functionality.
Frontend || https://task-manager-app-f8n7.vercel.app
Backend || https://task-manager-app-jish.onrender.com

✨ Features
Authentication
User Signup & Login
JWT Authentication

Role-based Access (Admin/User)
Admin Features
Create & Assign Tasks
Manage Team Members
View Dashboard Analytics
Track Task Status
Generate Reports
Calendar Task Management

Calendar View
